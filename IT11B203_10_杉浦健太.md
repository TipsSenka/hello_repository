# 月別予定管理カレンダー Webアプリ作成指示書

## 目的
HTML・CSS・JavaScriptを使用し、月ごとの予定を管理できるカレンダーWebアプリを作成する。

---

# 必須事項

## 1. カレンダー表示

- 月単位でカレンダーを表示すること
- 1週間を日曜日～土曜日で表示すること
- 当月の日付を正しく配置すること
- 前月・次月へ移動できること

## 2. 予定管理機能

- 日付をクリックして予定を登録できること
- 1日に複数の予定を登録できること
- 予定を編集できること
- 予定を削除できること
- 登録された予定をカレンダー上に表示すること

## 3. データ保存

- LocalStorageを使用すること
- ページを再読み込みしても予定が保持されること

## 4. デザイン

- レスポンシブ対応
- スマートフォンでも利用しやすいこと
- 現在の日付を強調表示すること
- シンプルで見やすい配色にすること

## 5. コード品質

- HTML
- CSS
- JavaScript

を分離して記述すること。

ファイル構成は以下とすること。

```text
index.html
style.css
script.js
```

---

# 作成手順

## Step1

基本レイアウトを作成

- タイトル
- カレンダー表示エリア
- 月変更ボタン

を配置する。

---

## Step2

JavaScriptでカレンダー生成機能を実装する。

実装内容

- 現在年月取得
- 月初取得
- 月末取得
- 曜日計算
- 日付マス生成

---

## Step3

月移動機能を実装する。

実装内容

- 前月ボタン
- 次月ボタン

クリック時に再描画する。

---

## Step4

予定登録機能を実装する。

実装内容

- 日付クリック
- 入力ダイアログ表示
- 予定保存

---

## Step5

LocalStorage保存機能を実装する。

保存形式例

```json
{
    "2026-07-09": [
        "課題提出",
        "アルバイト"
    ]
}
```

---

## Step6

予定表示機能を実装する。

実装内容

- 該当日付の予定表示
- 複数件対応
- 省略表示対応

---

## Step7

予定編集・削除機能を実装する。

実装内容

- 編集ボタン
- 削除ボタン
- 保存処理更新

---

## Step8

レスポンシブデザインを実装する。

実装内容

- PC表示
- タブレット表示
- スマホ表示

に対応すること。
