# アプリケーション開発指示書

## プロジェクト概要

### アプリケーション名

Gym Dashboard

### 概要

ジムでのトレーニングをサポートするダッシュボード風Webアプリケーションを作成する。

休憩タイマー、トレーニングメニュー、連続トレーニング日数、水分補給量などを1つの画面で管理できるようにする。

HTML・CSS・JavaScriptのみで実装し、サーバーやデータベースは使用しない。

---

# 開発ルール

* HTML・CSS・JavaScriptのみで実装する
* 外部ライブラリは使用しない
* ES6以降のJavaScriptを使用する
* コードには適度にコメントを書く
* 可読性・保守性を重視する
* ファイル構成は以下とする

```
/
├── index.html
├── style.css
└── script.js
```

---

# 必須機能

## 1. 今日のトレーニング

本日のトレーニングメニューを表示する。

表示内容

* トレーニング部位
* 種目名
* セット数
* 回数

メニューはJavaScriptの配列から表示すること。

---

## 2. 休憩タイマー

### 基本仕様

* 30秒
* 60秒
* 90秒

を選択できるようにする。

### ボタン

* Start
* Pause
* Reset

### タイマー表示

以下の形式で表示する。

```
01:30
```

### タイマー終了

* アラートを表示する
* 「トレーニング再開！」と表示する

---

## 3. トレーニングStreak

連続トレーニング日数を表示する。

例

```
🔥 15 Days
```

ブラウザのLocalStorageを利用して保存する。

---

## 4. 水分補給

水分補給量を管理する。

ボタン

* +100ml
* +200ml
* +500ml

現在の合計摂取量を表示する。

---

## 5. モチベーション

ページを開くたびにランダムで筋トレ名言を表示する。

例

* No Pain No Gain
* Train Hard
* Stay Strong
* Never Give Up

---

# データ保存

ブラウザのLocalStorageを使用する。
保存する内容

* Streak
* 水分補給量
* 最後に選択したタイマー時間

ページを再読み込みしても保持されること。

# JavaScript仕様

以下のように関数を分けること。

```
loadWorkout()
startTimer()
pauseTimer()
resetTimer()
updateTimer()
updateWater()
saveData()
loadData()
showMotivation()
```

イベントリスナーは最後にまとめて記述すること。

---

# エラーハンドリング

以下に対応する。

* タイマーの二重起動を防ぐ
* LocalStorageにデータが存在しない場合でもエラーにならない
* タイマー終了後は自動で停止する

# コーディング規約

変数名・関数名は英語。

例

```
timer
timerInterval
remainingTime
waterAmount
streak
workoutList
```

定数はconstを使用する。
再代入するもののみletを使用する。
varは禁止する。

# 完成条件

以下を満たしたら完成。

* 全機能が正常に動作する
* コンソールエラーが発生しない
* リロード後もデータが保持される
* レスポンシブ表示が崩れない
* HTML・CSS・JavaScriptのみで動作する
