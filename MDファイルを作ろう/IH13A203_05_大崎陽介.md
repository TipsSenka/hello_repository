# 累乗（倍々）計算アプリ PoC 仕様書

JavaScriptで動作する簡易的な自動計算・表示アプリケーションのPoC（概念実証）開発指示書です。

---

## 1. 概要
画面のロード完了と同時に、初期値「1」からスタートし、2秒ごとに「前回の計算結果 + 前回の計算結果」（2倍の計算）を自動で繰り返し、結果をリアルタイムで画面上に更新表示するWebアプリケーション。

---

## 2. 実装仕様

### 2.1 計算ロジック
1. **初期状態**: 数値 `1` 
2. **初回計算（ロード直後または最初のタイマー発火時）**: `1 + 1 = 2`
3. **以降の計算（2秒周期）**: `前回の結果 + 前回の結果`（例: `2 + 2 = 4`, `4 + 4 = 8`, `8 + 8 = 16` ...）
4. **タイマー間隔**: 2,000ミリ秒（2秒）

### 2.2 UI / フロントエンド
- **画面要素**:
  - 現在の計算結果を表示するメインエリア（視認しやすい大きめのフォント）
  - 現在のステップ数（何回目の計算か）や計算式（例: `16 + 16 = 32`）を表示するサブエリア（任意・推奨）
- **表示更新**: 計算が行われるたびに、画面上の表示を即座に書き換える（DOM操作）。

## 5. 開発における注意点・留意事項

### 巨大な数値への対応（オーバーフロー）
JavaScriptの標準数値型（`Number`）は、最大安全整数である **2^53 - 1** （`9,007,199,254,740,991`）を超えると精度が落ちる性質があります。  
長時間の連続動作や、それを超える数値を扱うことが想定される場合は、**`BigInt` の使用を検討**してください。

### 実行制御
本開発は **PoC（概念実証）レベル**のため、「一時停止/再開ボタン」などの細かなUI・機能実装は必須としません。  
ただし、今後のフェーズで機能追加がスムーズに行えるよう、**コードの拡張性を考慮した設計**を心がけてください。
