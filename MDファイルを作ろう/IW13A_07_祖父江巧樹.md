# JavaScriptで世界時計を作ってほしい

## 目標
JavaScriptを使って、地球儀の上に刺した場所の現在時刻を表示する世界時計を作ります。

## できるようにしたいこと
- 地球儀の画像や図形の上に、場所を表すマーカーを配置する
- マーカーをクリックすると、その場所の現在時刻を表示する
- 1秒ごとに時刻が更新される
- クリックした場所によって表示する時間が変わる
- 地球儀らしい見た目で、個性的な仕上がりにする

## 使う要素
- HTML: 地球儀と時計表示の領域を作る
- CSS: 地球儀の見た目やマーカーの配置を整える
- JavaScript: 現在時刻を取得し、選ばれた場所の時差を計算して表示する

## 手順
1. HTMLで地球儀の表示領域を作る
2. CSSで地球儀の形やマーカーの見た目を整える
3. JavaScriptのDateオブジェクトを使って現在時刻を取得する
4. 地球儀上の各マーカーに都市名や時差を設定する
5. マーカーをクリックしたとき、その場所の時刻を表示する
6. setIntervalを使って1秒ごとに時刻を更新する
7. 見た目を地球儀っぽくして、個性を出す

## ヒント
- 日本はUTC+9です
- ロンドンはUTC+0です
- ニューヨークはUTC-5または-4です
- 地球儀の場所ごとに時差を考えて表示しましょう

## 完成イメージ
- 地球儀の上に「東京」「ロンドン」「ニューヨーク」などのマーカーが刺さっている
- マーカーをクリックすると、その場所の現在時刻が表示される
- 時計がリアルタイムで動き、見た目も地球儀らしい

## サンプルデータ
- 東京: UTC+9
- ロンドン: UTC+0
- ニューヨーク: UTC-5または-4
- シドニー: UTC+10
- パリ: UTC+1

## 例のデータ構造
```js
const cities = [
  { name: "東京", utcOffset: 9 },
  { name: "ロンドン", utcOffset: 0 },
  { name: "ニューヨーク", utcOffset: -5 },
  { name: "シドニー", utcOffset: 10 }
];
```
