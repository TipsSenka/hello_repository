# JavaScriptで五目並べを作る

JavaScriptで遊べる五目並べゲームを作るための指示書です。クリックで石を置き、先に5個そろえた方が勝ちになるように作成します。

## 完成イメージ

- 15×15の盤面を表示する。
- 黒と白の石を交互に置ける。
- 5個連続でそろうと勝敗を判定する。
- 勝敗が決まったらゲームを止める。
- リセットボタンで最初からやり直せる。

## 作る内容

1. HTMLで盤面とボタンを用意する。
2. CSSで見やすい盤面と石のデザインを作る。
3. JavaScriptでクリックされたマスに石を置く。
4. 黒と白の手番を交互に切り替える。
5. 縦・横・斜めの5連続を判定する。
6. 勝敗表示とリセット処理を追加する。

## 作成手順

1. まずHTMLでタイトル、盤面、メッセージ欄、リセットボタンを配置する。
2. 次にCSSで盤面を正方形にそろえ、黒石・白石が見やすくなるように整える。
3. JavaScriptで盤面データを2次元配列で作る。
4. マスをクリックしたら、その位置に現在の手番の石を置く。
5. 置いたあとに勝敗判定を行い、5個そろっていれば終了する。
6. リセットボタンで盤面と手番を初期状態に戻す。

## 必須事項

- 15×15の盤面を表示すること。
- クリック操作で石を置けること。
- 黒と白の手番が交互に切り替わること。
- 5個並んだときに勝敗が判定されること。
- 勝敗が出た後は操作できないこと。
- リセット機能があること。

## 機能一覧

- 盤面を15×15マスで表示する。
- マスをクリックして石を置ける。
- 黒と白が交互に打てる。
- 5個並んだら勝ちを判定する。
- 勝敗メッセージを表示する。
- リセットで最初の状態に戻せる。

## ポイント

- 盤面は2次元配列で管理すると作りやすいです。
- 勝敗判定は、置いたマスを起点に4方向を調べると分かりやすいです。
- ゲーム終了後は、これ以上石を置けないようにします。
