#JavaScriptで可愛いピンクの時計を作ってほしい

## 作成手順（指示書）

以下は、このプロジェクトを作成・動作させるための手順です。`cute-pink-clock` フォルダを作り、必要なファイルを配置してください。

### 準備と必要ファイル
- フォルダ名: `cute-pink-clock`
- 必要ファイル:
	- `index.html` — HTML（時計の構造）
	- `styles.css` — デザイン（ピンク系のスタイル、目盛り、数字）
	- `script.js` — 時刻を取得して針とデジタル表示を更新するスクリプト
	- `README.md` — 起動方法とカスタマイズの説明

### ディレクトリ構成（例）
```
cute-pink-clock/
	index.html
	styles.css
	script.js
	README.md
```

### 手順
1. ワークスペース内に `cute-pink-clock` フォルダを作成します。
2. 上記の4つのファイルを作成し、それぞれにコードを記述またはコピーします（既に同名ファイルがある場合は上書きしないよう注意）。
3. ブラウザで動作確認します。方法は次の通りです。

#### 簡単な起動方法（ファイルを直接開く）
- エクスプローラーで `cute-pink-clock/index.html` をダブルクリックしてブラウザで開きます。

#### ローカルサーバーで開く（推奨）
- ターミナルでワークスペースのルートに移動して、以下を実行します（Pythonが使える場合の例）。

```powershell
cd "c:\Users\mizutani\Desktop\HAL\授業外\専科\hello_repository"
python -m http.server 8000
# ブラウザで以下を開く
# http://localhost:8000/cute-pink-clock/index.html
```

### 各ファイルの役割と注意点
- `index.html`: 時計のHTML。`<div>` に針（hour/minute/second）と数字・目盛りを入れる。
- `styles.css`: 色変数（:root）でピンク系の色を管理するとカスタマイズが簡単です。レスポンシブ対応を入れるとスマホでも見やすくなります。
- `script.js`: setInterval で1秒ごとに時間を取得して針を回転させ、デジタル表示（HH:MM:SS）を更新します。初期表示をupdateしてからintervalを回すと表示のチラつきが減ります。
- `README.md`: 開き方、編集箇所（色／サイズ）やよくあるカスタマイズ案を記載してください。

### カスタマイズ案（簡単で効果的）
- 色を変える: `styles.css` の `--pink` / `--accent` を編集
- フォント変更: Google Fonts を読み込んでタイトルや数字を可愛くする
- 目盛りの強調: 3時/6時/9時/12時の数字サイズを大きくする
- 12/24 時間表示切替: `script.js` にフラグを追加してデジタル表示を切替

### 問題が起きたとき
- ブラウザのコンソールを開いてエラーを確認してください（F12）。
- ファイルを配置しても動かない場合は、ローカルサーバーで開いてパスが正しいかを確認してください。
