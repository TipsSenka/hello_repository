# JavaScriptでおみくじアプリを作ってほしい

このREADMEは、Markdownの書き方を練習するためのサンプルです。  
JavaScriptを使って、ボタンを押すとランダムでおみくじの結果が表示される簡単なアプリを作ります。

## 作り方

1. 新しいHTMLファイルを作る
2. 画面にタイトル、結果表示エリア、ボタンを用意する
3. JavaScriptでおみくじの結果を配列に入れる
4. ボタンがクリックされたら、ランダムで結果を表示する

## サンプルコード

```html
<!doctype html>
<html lang="ja">
<head>
	<meta charset="UTF-8" />
	<meta name="viewport" content="width=device-width, initial-scale=1.0" />
	<title>おみくじアプリ</title>
	<style>
		body {
			margin: 0;
			min-height: 100vh;
			display: grid;
			place-items: center;
			background: #f5f7fb;
			font-family: "Yu Gothic", sans-serif;
		}

		.container {
			text-align: center;
			background: white;
			padding: 40px;
			border-radius: 16px;
			box-shadow: 0 8px 24px rgba(0, 0, 0, 0.12);
		}

		h1 {
			margin-bottom: 24px;
			color: #1f2937;
		}

		#result {
			font-size: 40px;
			font-weight: bold;
			margin: 24px 0;
			color: #dc2626;
		}

		button {
			font-size: 18px;
			padding: 12px 24px;
			border: none;
			border-radius: 8px;
			background: #2563eb;
			color: white;
			cursor: pointer;
		}

		button:hover {
			background: #1d4ed8;
		}
	</style>
</head>
<body>
	<div class="container">
		<h1>今日のおみくじ</h1>
		<div id="result">?</div>
		<button id="drawButton">おみくじを引く</button>
	</div>

	<script>
		const results = ["大吉", "中吉", "小吉", "吉", "末吉", "凶"];
		const resultElement = document.getElementById("result");
		const drawButton = document.getElementById("drawButton");

		drawButton.addEventListener("click", function () {
			const randomIndex = Math.floor(Math.random() * results.length);
			resultElement.textContent = results[randomIndex];
		});
	</script>
</body>
</html>
```

## ポイント

- `const results = [...]` でおみくじの結果を配列として用意する
- `Math.random()` でランダムな数字を作る
- `Math.floor()` で小数を切り捨てて整数にする
- `addEventListener("click", ...)` でボタンを押した時の処理を書く
- `textContent` を使って、画面に表示する文字を変更する
