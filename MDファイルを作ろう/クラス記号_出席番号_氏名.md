#JavaScriptで時計を作ってほしい

## 作り方

1. 新しいHTMLファイルを作る
2. 画面に時計を表示するための要素を用意する
3. JavaScriptで現在時刻を取得して表示する
4. 1秒ごとに表示を更新する

## サンプルコード

```html
<!doctype html>
<html lang="ja">
<head>
	<meta charset="UTF-8" />
	<meta name="viewport" content="width=device-width, initial-scale=1.0" />
	<title>デジタル時計</title>
	<style>
		body {
			margin: 0;
			min-height: 100vh;
			display: grid;
			place-items: center;
			background: #f5f7fb;
			font-family: "Yu Gothic", sans-serif;
		}

		#clock {
			font-size: 56px;
			font-weight: bold;
			letter-spacing: 0.08em;
			color: #1f2937;
		}
	</style>
</head>
<body>
	<div id="clock">00:00:00</div>

	<script>
		function formatTime(value) {
			return String(value).padStart(2, "0");
		}

		function updateClock() {
			const now = new Date();
			const hour = formatTime(now.getHours());
			const minute = formatTime(now.getMinutes());
			const second = formatTime(now.getSeconds());

			document.getElementById("clock").textContent = `${hour}:${minute}:${second}`;
		}

		updateClock();
		setInterval(updateClock, 1000);
	</script>
</body>
</html>
```

## ポイント

- `new Date()` で今の時刻を取得する
- `padStart(2, "0")` で1桁の数字を2桁表示にする
- `setInterval(updateClock, 1000)` で1秒ごとに更新する
