# JavaScriptで画面中央に0.5秒ごとに2倍する数字を表示してほしい

## 作り方

1. 新しいHTMLファイルを作る
2. 画面中央に数字を表示するための要素を用意する
3. 最初の値を `1` にする
4. `0.5秒` ごとに今の値を `2倍` して表示を更新する

## 動作イメージ

```
1 → 2 → 4 → 8 → 16 → 32 → ...
```

- 表示は画面の中央
- 更新間隔は `500ミリ秒`
- 止めずに増え続けてよい

## サンプルコード

```html
<!doctype html>
<html lang="ja">
<head>
	<meta charset="UTF-8" />
	<meta name="viewport" content="width=device-width, initial-scale=1.0" />
	<title>2倍カウンター</title>
	<style>
		body {
			margin: 0;
			min-height: 100vh;
			display: grid;
			place-items: center;
			background: #f5f7fb;
			font-family: "Yu Gothic", sans-serif;
		}

		#number {
			font-size: 72px;
			font-weight: bold;
			color: #1f2937;
		}
	</style>
</head>
<body>
	<div id="number">1</div>

	<script>
		let value = 1;
		const numberElement = document.getElementById("number");

		function updateNumber() {
			value *= 2;
			numberElement.textContent = value;
		}

		setInterval(updateNumber, 500);
	</script>
</body>
</html>
```

## ポイント

- `display: grid` と `place-items: center` で画面中央に表示する
- 最初の値は `1` にする
- `value *= 2` で現在の値を2倍する
- `setInterval(updateNumber, 500)` で0.5秒ごとに更新する
- 表示する要素の `id` と `getElementById` の名前をそろえる
